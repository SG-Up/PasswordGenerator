# PwdGen
Password Generator

A small password generator based on old 2000's keygen aesthetics. Uses RNGCryptoServiceProvider over a standard RNG generator.

## USE ##

1) Enter length of password
2) Select if special characters are needed
3) Click generate

## OPTIONAL ##

1) Click mute to mute bckground music
2) Click copy to copy password to clipboard
3) Click clear to clear password box
